# Landing Page – División Económico-Administrativa

### Universidad Tecnológica de Cancún

## Descripción del Proyecto

Este repositorio contiene la **landing page de la División Económico-Administrativa**, desarrollada como parte de una actividad académica.
El proyecto consiste en una página principal (**index.html**) construida utilizando **HTML5**, **CSS** y el framework **Bootstrap**, con el objetivo de presentar de forma clara y estética la información de la división, sus programas académicos y servicios.

## Objetivo

Diseñar y estructurar una página web informativa que represente visualmente a la División Económico-Administrativa, aplicando principios de diseño web responsivo y buenas prácticas en el uso de Bootstrap.

## Tecnologías Utilizadas

* **HTML5**
* **CSS**
* **Bootstrap 5**
* **Git & GitHub** (control de versiones)
